# Playbook — New Feature (Dalam Milestone Existing)

## Kapan dipakai
Fitur jelas, dalam milestone yang sudah berjalan, tidak perlu PRD baru.

## Flow
BARRY QD → /gsd:plan-phase → /gsd:execute-phase

## Steps
1. Brief Barry QD: describe fitur dalam 1-2 paragraf
2. Barry produce technical spec + task breakdown
3. /gsd:plan-phase [phase] — buat PLAN.md
4. /clear → sesi baru
5. /gsd:execute-phase [phase] — eksekusi

## Tips
- Kalau scope tidak jelas → BMAD John dulu (PRD)
- Kalau ada UI → /gsd:discuss-phase dulu untuk design
- Commit per task, bukan per phase
